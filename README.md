# Portfolio (html)

Personal developer portfolio built with **HTML, CSS, and vanilla JavaScript**.

Live site:  
https://andreibenea.github.io/portfolio-html/

---

## Overview

This repository contains the source for my portfolio website, including:

- **Landing page** (`index.html`) with navigation to projects and contact info  
- **About Me page** with a built-in **Resume section**  
- **Project pages** under `/pages/` with write-ups and screenshots  
- **Playable demos** under `/games/` (e.g. *HeliStrike*, *C-Eleven*)  
- **Media and assets** in `/images/` and `/media/`  
- Supporting scripts and styles:
  - `style.css`
  - `script.js`

The focus of the project is clean, responsive layout and clear presentation of my work and experience.
