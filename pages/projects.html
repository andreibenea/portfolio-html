Welcome to My Projects Page!

On this page, you'll find a collection of the projects I've worked on, showcasing my skills and creativity. Each project reflects my passion for problem-solving, design, and development. These projects span various aspects of web development and software design, and I'm excited to share them with you.

Take a look at the details below, and feel free to explore each project to learn more about how it was built and the challenges I overcame. If you have any questions or feedback, I'd love to hear from you!

Project 1: Kreativstorm
A sample website created to demonstrate front-end skills using HTML, CSS, and JavaScript. Ideal for showcasing responsive design and presentation.
Project 2: Calculator
A simple calculator built with JavaScript, highlighting key concepts like DOM manipulation and event handling.
Project 3: Rock-Paper-Scissors
A basic Rock-Paper-Scissors game using JavaScript and alert prompts to handle user interaction.
Project 4: Shopping Manager
A Python-based shopping list manager showcasing data handling and file management in a simple application.
Project 5: Digital Recipe Book
A Django web app that lets users add, view, and manage recipes. It includes user login, organized recipe listings, and a clean, easy-to-use interface.

Note: The embedded version is still being finalized — login inside the iframe is temporarily unavailable due to CSRF settings. You can explore the live site directly at sdevgroupsix.pythonanywhere.com. If you'd like to test the app, please use the contact form on this site to request access credentials.

Project 6: HeliStrike
A 2D helicopter shooter game built in Godot. It features smooth controls, infinite scrolling, and sound effects for an engaging arcade-style experience.
